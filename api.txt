VEHICLE APIs 

1. NHTSA VEHICLE API

OFFICIAL LINK:
https://vpic.nhtsa.dot.gov/api/

When we use the NHTSA Vehicle API, we can find official and verified vehicle
information provided by the U.S. government. 

Information available:
- VIN decoding
- Vehicle make, model, and year
- Manufacturer details
- Body type and vehicle class
- Safety ratings
- Recall and defect information
- Consumer complaints



2. DATA.GOV VEHICLE DATASETS

OFFICIAL LINK:
https://catalog.data.gov/dataset/?tags=vehicles

From Data.gov vehicle datasets, we can find statistical and analytical vehicle
data published by government agencies.

Information available:
- Vehicle registration statistics
- Fuel type and emissions data
- Electric and hybrid vehicle data
- Transportation and traffic-related statistics



3. EDMUNDS VEHICLE API (LEGACY)

OFFICIAL LINK:
https://developer.edmunds.com

The Edmunds API (legacy) provides detailed commercial vehicle data.


Information available:
- Vehicle specifications and trims
- Pricing and MSRP details
- Ownership and maintenance cost estimates

This source is used as a reference for pricing and specification comparison.


4. OPENDATASOFT VEHICLE DATASETS

OFFICIAL LINK:
https://public.opendatasoft.com/

From OpenDataSoft vehicle datasets, we can access open reference data related to
vehicle models and specifications.

Information available:
- Vehicle model and manufacturer data
- Fuel type and basic technical details
- Engine and transmission information

This data is useful for validating vehicle model and specification details
mentioned in contracts.
