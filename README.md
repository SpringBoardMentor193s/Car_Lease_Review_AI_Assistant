# Car_Lease_Review_AI_Assistant
The Car Lease or Loan Contract Review and Negotiation App is an AI-driven mobile application designed to assist consumers in understanding, reviewing, and negotiating their car lease or loan contracts. The application leverages Large Language Models (LLMs) for SLA (Service Level Agreement) extraction, identifies key contract terms.
## Project Overview
Car Lease Review AI Assistant is an AI-powered application designed to help users understand, analyze, and negotiate car lease or loan contracts using Large Language Models.

## Objectives
- Simplify complex car lease and loan agreements
- Identify hidden clauses and risks
- Explain contract terms in simple language
- Help users make informed financial decisions

## Key Features
- Contract text analysis
- Clause extraction and explanation
- AI-assisted negotiation support
- User-friendly interface (planned)
